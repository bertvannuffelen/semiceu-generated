Dieses Dokument beschreibt ein Anwendungsprofil, in diesem Fall TertiaryEducationDiplomaEvidence-AP-DE. Das Anwendungsprofil spezifiziert ein semantisches Datenmodell, welches eine Reihe von Anwendungsfällen abdeckt. Die Spezifikation besteht aus Begriffen mit ihren Einschränkungen (Kardinalitäten, Codelisten, Nutzungsrichtlinien).
