This document describes a vocabulary; in this case TertiaryEducationDiplomaEvidence-voc-EN. A vocabulary is a collection of reusable terms, more precisely classes and properties, of which their semantics is fixed by means of a label and a definition. For terms definied in this domain additional formal agreements may be defined. The formal agremeents for external terms, reused in this domain, are adopted from that external source.

Deze sectie geeft een formele definitie aan elke klasse.

Class Address

Type Class
URI https://sdg.semic.euAddress
Definition An "address representation" as conceptually defined by the INSPIRE Address Representation data type.

Class Course Result

Type Class
URI https://sdg.semic.euCourseResult
Definition Grade obtained after finishing/completing a course.

Type Class
URI https://sdg.semic.euEducationInstitution
Specialisatie van
  • https://sdg.semic.euOrganisation
Definition An Organisation that provides instructional services to individuals or education-related services to individuals and other educational institutions.

Class Grade

Type Class
URI https://sdg.semic.euGrade
Definition Mark indicating a degree of accomplishment.

Class Location

Type Class
URI https://sdg.semic.euLocation
Definition A spatial region or named place.

Class Organisation

Type Class
URI https://sdg.semic.euOrganisation
Definition Represents a collection of people organized together into a community or other social, commercial or political structure. The group has some common purpose or reason for existence which goes beyond the set of people belonging to it and can act as an Agent

Class Person

Type Class
URI https://sdg.semic.euPerson
Definition An individual person who may be dead or alive, but not imaginary.

Class Record of Results Evidence

Type Class
URI https://sdg.semic.euRecordOfResultsEvidence
Definition A record of the Student's progress in his/her studies: the educational components they have taken, the number of ECTS credits they have achieved, and the grades they have been awarded.

Class Student

Type Class
URI https://sdg.semic.euStudent
Specialisatie van
  • https://sdg.semic.euPerson
Definition A learner enrolled on a formal educational programme at a tertiary Education Institution.

Class Tertiary Education Diploma Evidence

Type Class
URI https://sdg.semic.euTertiaryEducationDiplomaEvidence
Specialisatie van
  • https://sdg.semic.euTertiaryEducationEvidence
Definition Any formally awarded qualification/credential, issued by a competent authority attesting the successful completion of a recognised programme of study of tertiary education.

Class Tertiary Education Diploma Supplement Evidence

Type Class
URI https://sdg.semic.euTertiaryEducationDiplomaSupplementEvidence
Definition Document accompanying a Tertiary Education Diploma Evidence, providing a standardised description of the nature, level, context, content and status of the studies completed by its holder.

Class Tertiary Education Evidence

Type Class
URI https://sdg.semic.euTertiaryEducationEvidence
Definition Abstract superclass for evidences that are issued after obtaining a Tertiary Education grade.
